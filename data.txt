The dataset is inspired by the Dichasus-cf0x dataset.

Noise was added to each dataset both in the csi features as well as the positions target variable. This was done independently for each observation.

The observations that are labeled as is_forget = 1 are observations that are much noisier than others and therefore is data we dont want to retain in the model. For this though we must not retrain the model but do machine unlearning.
The following is information on the datasets that you have:

Files
Training set: This dataset is included in the data section as:

task2_train_metadata.csv - includes meta data on the observations. This includes the is_forget label. Specifically has: sample_index, sequence_id, relative_time, is_forget
task2_train_csi.npy - includes the "csi" information on each observation. Size of [10918, 2, 32, 64]
task2_train_positions.npy - includes the target positions to predict. Size of (10918, 3).
Test set:

task2_test_csi.npy - includes the "csi" information on each observation for the test set. [2723, 2, 32, 64]
task2_test_positions.npy - includes the target positions to predict for the test set. Size of (2723, 3)
task2_test_metadata.csv - includes the test set metadata without the is_forget labels.
Other files:

baseline_cnn_task2.pth - contains the pre-trained model checkpoint.
competition_notebook.ipynb - contains code for the Pytorch model architecture, fine-tuning code, evaluation code, and data pre-processing.
NOTE: The competition_notebook.ipynb is ready to use in the google colab environment for everyone. A requirement of this though is that everyone chooses a GPU in the runtime. For this, do the following:

Go to change runtine type on the top right next to the RAM and Disk metrics.
Select the hardware accelerator to T4 GPU.
